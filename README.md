# react-marvel

Simple React-Redux App using the Marvel API

Concept in this small app :

- [x] react-router
- [x] redux
- [x] reducers
- [x] actions
- [x] class components
- [x] functions components
- [x] use of states and Redux state
- [x] axios for AJAX

# install

```
You can Git (clone) or download the zip file
inside of the folder :

npm install
npm start

that's it
```
